# FitLife
# 💪 FitLife — Aplikacja Fitness

> Mobilna aplikacja do śledzenia treningów, kalorii i postępów fitness.

---

## 📱 O Projekcie

**FitLife** to aplikacja mobilna stworzona w React Native, zaprojektowana, aby pomóc użytkownikom osiągać cele fitness. Aplikacja umożliwia planowanie treningów, śledzenie spożycia kalorii, monitorowanie postępów oraz motywuje poprzez przejrzysty i intuicyjny interfejs.

Projekt został stworzony w ramach kursu programowania mobilnego. Prototyp UI/UX zaprojektowano w Figmie.

---

## ✨ Funkcjonalności

### 🏋️ Treningi i Ćwiczenia
- Przeglądanie biblioteki ćwiczeń z opisami i instrukcjami
- Tworzenie własnych planów treningowych
- Zapisywanie wykonanych treningów (serie, powtórzenia, ciężary)

### 🔥 Śledzenie Kalorii
- Rejestrowanie posiłków i spożycia kalorii
- Podgląd makroskładników (białka, tłuszcze, węglowodany)
- Ustawianie dziennych celów kalorycznych

### 📊 Postępy i Statystyki
- Wykresy historii treningów
- Śledzenie masy ciała w czasie
- Podsumowania tygodniowe i miesięczne

### 📅 Harmonogram Treningów
- Planowanie treningów na cały tydzień
- Ustawianie przypomnień i powiadomień
- Widok kalendarza z nadchodzącymi sesjami
