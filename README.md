# 💪 FitLife — Aplikacja Fitness na iOS

> Mobilna aplikacja do śledzenia treningów, kalorii i postępów fitness.

---

## 📱 O Projekcie

**FitLife** to aplikacja mobilna zaprojektowana dla osób, które chcą prowadzić aktywny tryb życia i skutecznie osiągać swoje cele fitness. Aplikacja oferuje przejrzysty, nowoczesny interfejs w ciemnym motywie, który prowadzi użytkownika przez cały proces — od planowania treningu, przez śledzenie diety, aż po analizę postępów.

Projekt został stworzony w ramach kursu programowania mobilnego. Prototyp interfejsu zaprojektowano w narzędziu do projektowania UI zgodnie z wytycznymi iOS Human Interface Guidelines.

---

## ✨ Funkcjonalności

### 🏠 Strona Główna (Dashboard)
- Spersonalizowane powitanie z datą i porą dnia
- Kołowy wykres dziennego spożycia kalorii
- Szybki podgląd kroków, nawodnienia i snu
- Karta dzisiejszego treningu z paskiem postępu i przyciskiem Start

### 🏋️ Treningi
- Lista dostępnych planów treningowych
- Wyszukiwarka treningów
- Filtrowanie według kategorii: Wszystkie / Siłowe / Cardio / Joga
- Każdy trening zawiera czas trwania, spalone kalorie oraz poziom trudności (Łatwy / Średni / Trudny)

### 🥗 Żywienie
- Dzienny cel kaloryczny z paskiem postępu
- Podział makroskładników: Białko, Węglowodany, Tłuszcze
- Dziennik posiłków podzielony na: Śniadanie, Obiad, Kolacja, Przekąski
- Możliwość dodawania nowych posiłków

### 📊 Postępy
- Odznaka serii treningowej (np. 🔥 14 dni z rzędu)
- Tygodniowy wykres słupkowy aktywności
- Wykres śledzenia masy ciała w czasie
- Sekcja rekordów osobistych (najlepsze wyniki)

### 📅 Harmonogram
- Tygodniowy pasek kalendarza z podglądem dni
- Lista zaplanowanych treningów na wybrany dzień
- Podgląd treningu na następny dzień
- Możliwość dodawania nowych treningów do planu

### 👤 Profil
- Dane użytkownika z poziomem zaawansowania
- Statystyki: liczba treningów, utracone kilogramy, seria dni
- Ustawienia aplikacji: cele, powiadomienia, tryb ciemny
- Opcja wylogowania
